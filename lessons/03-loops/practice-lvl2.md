# Python Loop Exercises (Intermediate)

## 1. Print numbers until user types "stop" (while loop)
Keep asking the user for input and print it back.
Stop only when they type `"stop"`.

---

## 2. Count down from any number (while loop)
Ask the user for a number.
Use a `while` loop to count down to zero.

---

## 3. Sum only odd numbers from 1–100 (for + if)
Loop from 1 to 100.
Add only the odd numbers to a running total.

---

## 4. Print all letters except vowels (for + if-else)
Given a word, print each letter.
Skip vowels using an `if` check.

---

## 5. Keep asking for a password until correct (while + if)
Set a predefined password.
Loop until the user enters the correct one.

---

## 6. Print a square of stars (nested for)
Ask the user for a size (e.g., 5).
Print a square pattern using `*`.

---

## 7. Find the largest number in a list (for + if)
Given a list of numbers, loop through and track the largest value.

---

## 8. Count how many items are > 10 (for + if)
Given a list, count how many are greater than 10.

---

## 9. Ask user 5 times for numbers, sum only positive ones (for + if)
Loop exactly 5 times.
If the number is positive, add it to a running sum.

---

## 10. Loop menu (while + if-elif)
Create a loop that shows:
1. Print “Hello”
2. Print today’s date
3. Exit

Keep looping until user selects “3”.
