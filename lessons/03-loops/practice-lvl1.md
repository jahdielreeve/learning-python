# Python Loop Exercises (Beginner)

## 1. Print numbers 1 to 10  
Write a loop that prints the numbers 1 through 10.

---

## 2. Print even numbers up to 20  
Write a loop that prints only even numbers from 1 to 20.

---

## 3. Sum numbers 1–100  
Use a loop to calculate the sum of all numbers from 1 to 100.

---

## 4. Countdown from 10  
Create a loop that prints 10 down to 1.

---

## 5. Print each letter in a word  
Ask the user for a word, then print each letter one by one.

---

## 6. Reverse a word (using a loop only)  
Ask for a word and build its reverse manually using a loop.

---

## 7. Count how many times “a” appears  
Given a sentence, count how many times the letter `a` appears.

---

## 8. Multiplication table  
Ask the user for a number and print its multiplication table from 1 to 10.

---

## 9. Print only values greater than 5  
Given a list of numbers, print only those greater than 5.

---

## 10. Ask for 5 user inputs  
Ask the user to type something 5 times and print each response.
