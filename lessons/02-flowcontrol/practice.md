# Python If–Else Practice Problems (Beginner → Intermediate)

## 1. Even or Odd  
Ask the user for a number and tell them if it’s even or odd.

---

## 2. Age Category  
Ask for age.  
If < 13 → “child”,  
13–19 → “teen”,  
20–59 → “adult”,  
60+ → “senior”.

---

## 3. Positive, Negative, or Zero  
Ask the user for any number and classify it.

---

## 4. Pass or Fail  
Input a score out of 100.  
≥ 50 = “Pass”, otherwise “Fail”.

---

## 5. Compare Two Numbers  
Ask for two numbers.  
Tell which one is bigger, or if they are equal.

---

## 6. Temperature Check  
Ask for temperature in °C.  
If > 37.5 → “fever”, else → “normal”.

---

## 7. Discount Calculator  
Ask for purchase amount.  
If ≥ 100 → apply 10% discount.  
Else → no discount.

---

## 8. Username & Password Check  
Ask for username and password.  
If both match predefined values → “Login success”.  
Else → denied.

---

## 9. Grade Letter  
Input score.  
90+ → A  
80+ → B  
70+ → C  
60+ → D  
Else → F

---

## 10. Check if word contains “a”  
Ask for a word.  
If it contains letter `a` → print “has a”.  
Else → print “no a”.

---

## 11. Movie Ticket Pricing  
Ask age.  
If < 3 → free  
3–12 → child price  
13–59 → adult price  
60+ → senior price

---

## 12. Leap Year Check  
Given a year, determine if it’s a leap year.

---

## 13. Number in Range  
Ask for a number.  
Check if it’s between 1 and 100 (inclusive).

---

## 14. Divisible by 3 AND 5  
Check if a number is divisible by both 3 and 5.

---

## 15. Login Attempts (if + counter)  
User has 3 tries to enter the correct password.  
After 3 wrong tries → “locked”.
